Lenguaje C

El lenguaje de programación C es "el papi" de los lenguajes actuales de programación.

Python, Java, Javascript, Kotlin, GO y muchos lenguajes de programación actuales tienen su raíz de creación en el lenguaje C, en algunos es más notorio que en otros.

C es un lenguaje compilado, esto significa que es necesario crear un archivo distinto al código para poder correr el programa.
